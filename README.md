ubcv-locations
==============

Locations on UBC Vancouver Campus.

Files Provided
--------------
* Provided in geojson, csv, and file geodatabase formats.
* FGDB is ESPG:26910/UTM10N, geojson ESPG:4326.
* csv data has lat/long columns.
